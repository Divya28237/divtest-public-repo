Getting started with Nutella is easier than spreading it on toast:

This guide will help you clone the Nutella repo, install dependencies, and get the app running locally in a few simple steps.

# $livedoc$ Getting started with Nutella project setup

# Includes cloning, installing dependencies, and running the app

# Applies to all environments
 
echo "Getting started with Nutella is easier than spreading it on toast:"
 
# $livedoc$ Clone the repository

git clone https://github.com/YourOrg/Nutella.git
 
# $livedoc$ Navigate to the Nutella directory

cd Nutella
 
# $livedoc$ Install dependencies (if any)

npm install
 
# Important! This is one of the core commands to launch the project

npm start
